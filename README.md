#  COVID-19 Dashboard Analysis (India)

##  Project Overview
This project is an interactive data visualization dashboard built using Tableau to analyze COVID-19 trends in India. It helps in understanding the spread, impact, and key patterns of the pandemic through visual insights.

##  Objective
* To analyze COVID-19 data across Indian states
* To identify trends in cases, deaths, and recovery
* To present insights using interactive visualizations
* To make complex data easy to understand

##  Tools Used
* Tableau Public
* Microsoft Excel / CSV Dataset

##  Dashboard Features

###  State-wise Analysis
* Map visualization of total deaths across India
* Highlights most affected states

###  Age Group Distribution
* Donut chart showing case distribution among age groups

###  Gender Distribution
* Comparison of male vs female COVID-19 cases

###  Testing Infrastructure
* Bar chart showing number of ICMR labs per state

###  Vaccination Data
* Comparison of first and second doses administered

###  Trend Analysis
* Line chart showing:
 * Confirmed cases
  * Recovered cases
  * Death cases

### State-wise Testing
* Displays testing data across states

##  Key Insights
* Highly populated states show more cases and deaths
* Age group 20–39 is most affected
* Male cases are higher than female
* More testing leads to higher case detection
* First dose vaccination is higher than second dose
* COVID waves are clearly visible in trend graphs
